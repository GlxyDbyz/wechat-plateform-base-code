wechat-plateform-base-code
====
自己按照微信公众平API开发的微信公众平台相关代码完成了部分功能
`(Spring+SpringMVC+MyBatis+Durid)`<br><br>

#### 1.API接入
    a. 微信认证接入
    b.被动消息记录和回复
    `url`:http://[host]/[path]/app<br>
    `code`:AppController.java
    
#### 2.消息发送
    客服消息发送 (文本消息,图文消息)
    模版消息发送
    `code`:AppUtil.java
<br>

#### 3.菜单相关
    自定义菜单创建
    自定义菜单查询
    `code`:AppUtil.java
